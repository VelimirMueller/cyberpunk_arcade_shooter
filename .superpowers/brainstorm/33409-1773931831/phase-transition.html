Phase Transition Feel

When the boss hits an HP threshold and shifts phase, what should happen?

A

Stagger Window

Boss briefly freezes/staggers. Screen flash + glitch effect. Brief text like "PHASE SHIFT" appears. Boss resumes with new patterns. Player gets a 1-2 second breather. Classic Souls.

B

Rage Burst

Boss emits a shockwave that pushes player back. Screen shakes hard. Boss color/glow intensifies. No pause — the new phase hits immediately. The transition IS an attack. Aggressive, punishing.

C

Both — Boss Dependent

Some bosses stagger (Phantom, Sentinel — more methodical). Others rage burst (Berserker, Apex — aggressive). Matches each boss's personality. More work but more character.
